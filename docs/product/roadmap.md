# Product Roadmap

## Completed Features

- [x] **Player Management** — Register players, view profiles, track individual ELO ratings `M`
- [x] **Team System** — Form 2-player teams with automatic team ELO calculation `M`
- [x] **Match Recording** — Log matches with winner/loser and fanny flag support `S`
- [x] **Hybrid ELO Engine** — Individual ELO changes with K-factor tiers and pool correction `L`
- [x] **Rankings Leaderboards** — Player and team rankings sorted by ELO `S`
- [x] **Match History** — Paginated match history with ELO change tracking `M`
- [x] **Player Statistics** — Detailed stats pages with win rates and ELO progression `M`
- [x] **Team Statistics** — Team detail pages with match history and performance data `M`
- [x] **API Documentation** — Swagger UI and ReDoc for API exploration `XS`

> **Notes**
>
> - These features represent the current MVP, fully functional
> - Future features ordered by value-to-effort ratio and user impact
> - Each item is an end-to-end functional and testable feature
